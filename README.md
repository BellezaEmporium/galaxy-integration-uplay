# Ubisoft Connect (ex. Uplay) integration for GOG Galaxy

Allows to import your list of games, achievements and time played.
Forked from FriendsOfGalaxy.

## Information

The Galaxy 2.1 version of this plugin is available ! Please download the latest pre-release (which will always be called "Galaxy 2.1" in the end).
It is NOT meant to be used in a Galaxy 2.0 release, if you want to use it, please use the latest release file.

## Disclaimer

Please note that, in order to make this plugin fully functional, you need to have a valid Ubisoft Connect account.

## Disclaimer 2

While the plugin is proven to work on Windows, I *cannot confirm* that this plugin also works on Mac.

## Installation

1. Download the plugin from the [releases page](https://github.com/BellezaEmporium/galaxy-integration-uplay/releases). You'll have multiple versions, one for Windows, one for macOS.
2. Unzip the archive.
3. Extract the contents of the file into Ubisoft Connect's plugin location : uplay_afb5a69c-b2ee-4d58-b916-f4cd75d4999a (make sure to have downloaded the FriendsOfGalaxy version before putting the files in ! or else you'll be stuck in an endless loop).
4. Start GOG Galaxy, and enjoy !

## Configuration

Just login to your Ubisoft account from GOG Galaxy, and enjoy your games.
